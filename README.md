# schizoite0

This is my custom version of Bazzite, with a tinge of madness. Nothing is particularly "schizo" about this build other than its creator. The name is a portmanteau of "schizoid" and "Kinoite".

Differences from Bazzite:

* atop
* profile-sync-daemon
  * firefox-flatpak
  * waterfox-flatpak
  * vivaldi-flatpak
  * zen-flatpak
* prometheus-node-exporter
* yakuake

Many thanks to the Universal Blue and Fedora Project developers for making this possible 💙.

## Installation

```shell
sudo bootc switch ghcr.io/hyperreal64/schizoite0
```
